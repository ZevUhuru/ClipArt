# Esy Platform — Integration Design

Documentation for the Esy template engine and its integration with clip.art.

**Esy** is a template-to-artifact platform owned by ESY LLC. clip.art is a division of ESY LLC that consumes Esy's template runner API to produce structured artifacts (worksheets, flashcards, etc.) from generated clip art.

These docs live in the clip.art repo for convenience but describe the **API contract between the two products**. Esy has its own codebase, auth, and infrastructure.
